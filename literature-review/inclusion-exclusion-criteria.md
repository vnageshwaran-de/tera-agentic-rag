# Inclusion / Exclusion Criteria

Source: `literature-review/search-protocol.md` §6–§7.

## Inclusion criteria

A source is included if **all** of the following hold:

1. Publication date within 2021-01-01 through 2026-05-31, with pre-2021 foundational works admitted only as anchor citations.
2. Substantive technical content directly relevant to one or more of the seven research questions.
3. Source type is one of:
   - Peer-reviewed journal or conference paper,
   - Preprint with credible authorship and substantive technical claims,
   - Standards document or authoritative governance publication (NIST, OWASP, ISO/IEC, CSA),
   - Industry technical report with verifiable methodology.
4. Sufficient methodological detail for classification, comparison, or synthesis.
5. Published in English.

## Exclusion criteria

A source is excluded if **any** of the following hold:

1. Opinion piece without technical substance.
2. Vendor marketing material (used only as labeled industry context where unavoidable).
3. Outside the scope of the seven research questions.
4. Duplicate of an already-included entry per the deduplication procedure.
5. Retracted, withdrawn, or under public correction.
6. Predatory venue per the heritage Beall list and DOAJ exclusions.
7. Claims cannot be verified from accessible metadata, abstract, or legally available full text.
8. Non-English without an authoritative English translation.

## Source-type-specific handling

| Source type | Treatment |
|---|---|
| Peer-reviewed journal / conference | Primary evidence |
| Preprint (arXiv) | Used where peer-reviewed equivalent is unavailable; replaced when peer-reviewed version appears; labeled "preprint" in the manuscript |
| Standards / governance document | Treated as normative anchor, not empirical evidence |
| Industry technical report | Cited with explicit attribution; never the sole support for a quantitative claim |
| Vendor blog / whitepaper | Tier-5 context only; cited sparingly with explicit labeling |
